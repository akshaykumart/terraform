terraform scripting for creating aws resources.

## Commands
```
$ terraform init                    # initialize terraform environment and download plugins from provider.tf
$ terraform fmt                     # formats to hashicorp formats
$ terraform validate                # validates the terraform scripts
$ terraform plan                    # priview the changes to be made
$ terraform apply                   # apply the changes 
$ terraform apply -auto-approve     # apply the changes withot user prompt (yes)
$ terraform destroy                 # destroys the resources created
$ terraform destroy -auto-approve   # destroys the resources created without user prompt
```
